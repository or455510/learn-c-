For الحلقة التكرارية

تستخدم التكرارية للتعامل مع الكود أكثر من مرة مثل طباعة رسالة اكثر من مرة .

Syntax

for(statement1;statement2;statement3)

{

// code block to be executed

}

for(inti=0;i < 5;i++)

{

// code block to be executed

}

Statement1: تعريف متغير واعطائه قيمة ثابتة كأن تكون بداية للتكرار او نهاية

Statement2: شرط استمرار التكرار

Statement3: زيادة او نقصان استمرار التكرار

I++

فكلها نفس العمل i+=1 او استخدام i=i+1وتعني زيادة العدد قيمة واحدة ويمكن استخدام بدلها incrementوتسمى

المثال الاول

متى ما iواعطينا له قيمة ابتدائية وتساوي 0 , ووضعنا شرط التكرار طباعة قيمة iفي هذا المثال عرفنا متغير التكرار

زيادة بمقدار واحد i++ 1بقيمة iاصغر من 3 وزيادة , i كانت

{

Cout <<"i" << endl;

}

For الحلقة التكرارية المتداخلة

في كل حلقة تكرار واحدة يتم تنفيذ حلقة تكرارية متكاملة داخل هذه الحلقة الرئيسية

مثال الاول في هذا المثال حلقة متداخلة اي حلقة تكرارية داخلة حلقة اخرى

for(int i=1;i <= 3;i++)

{

for(int j=1;i <= 2;j++)

{

Cout <<"i=" i << "j=" j << endl;

}

}

Output :

i=1 j=2

i=2 j=1

i=2 j=2

i=3 j=1

i=3 j=2

Break and Continue التحكم بالعبارات التكرارية

وتستخدم لتوقف الحلقة التكرارية عند حد ما وذلك بوضع شرط فاذا تحقق الشرط ينهي الحلقة التكرارية عن الاستمرار . Break

for(int i=0;i < 10;i++)

{

if(i==4)

{

Break;

Cout << i <<"\n";

}

}

Output :

0

1

2

3
