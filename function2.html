الدوال الرياضية

على العديد من الوظائف التي تتيح لك أداء المهام الرياضية على الأرقام.c++ يحتوي

تقوم بارجاع القيمة الاكبر بين رقمينMax

Max (number1,number2);

تقوم بارجاع القيمة الاصغر بين رقمين Min

Min (number1,number2);

ومن هذه الدوال #include < cmath> المزيد من الوظائف الرياضية لكن تحتاج الى اضافة مكتبة c++كما توفر

لارجاع الجذر التربيعي للرقم Sqrt

Sqrt (number);

تقوم بتقريب الرقم الى اقرب عدد صحيح Round

Round (number);

لارجاع اللوغارتم الطبيعي للرقم Log

Log (number);

كما توجد العديد من الدوال الرياضية الاخرى يمكنك الاطلاع عليها

مثال كامل لكافة دوال الرياضية

#include <iostream>

#include <cmath>

using namespacestd;

int main(){

Cout << "Max" <<max(5,10); << endl;

Cout << "Min" <<Min(5,10); << endl;

Cout << "sqrt" <<sqrt(64); << endl;

Cout << "round" <<round(2.6); << endl;

Cout << "log" <<log(2);

Return 0;

}

Output :

Max:10

Min:5

Sqrt:8

Round:3

Log:0.693147
