C++ لكل لغة برمجة دالة خاصة بها لطباعة النصوص والارقام وقيم المتغيرات وعرضها على شاشة التنفيذ , وفي لغة

للمخرج والطباعة cout <<نستخدم ألاوبجكت

#include <iostream>

using namespace std;

int main(){

cout << "Hello World ";

return 0;

}

Output :

Hello World

طباعة سطر جديد

C++توجد طريقتين لطباعة سطر جديد في لغة

\n باستخدام

#include <iostream>

using namespace std;

int main(){

cout << "Hello World \n ";

cout << "I am learning c++ ";

return 0;

}

Output :

Hello World

I am learning c++

endl بأستخدام

#include <iostream>

using namespace std;

int main(){

cout << "Hello World " << endl;

cout << "I am learning c++ ";

return 0;

}

Output :

Hello World

I am learning c++

User Input المدخلات

وهي قيم يتم أدخالها من قبل المستخدم على شاشة التنفيذ ليتم معالجتها .

وهو المختص cin يستخدم لاخراج او طباعة القيم على شاشة التنفيذ فهنا سنتعرف على cout كما عرفنا سابقاً ان الاوبجكت

بادخال القيم على شاشة التنفيذ

>>هو متغير محدد مسبقًا يقرأ البيانات من لوحة المفاتيح باستخدام عامل الاستخراج.:Cin

مثال

x. ثم نطبع قيمة xفي هذا المثال ، يمكن للمستخدم إدخال رقم ، يتم تخزينه في المتغير

#include <iostream>

using namespace std;

int main(){

int x;

cout << "my age is:";

cin >>x;

cout << "your age is:" <<x;

return 0;

}

Output :

my age is:22

your age is:22

مثال جمع مدخلات رقمين

ومن ثم نقوم y والرقم الثاني في المتغير xفي هذا المثال سيقوم المستخدم بادخال رقمين يتم تخزين الرقم الاول في المتغير

sum وفي النهاية نقوم بطباعة sum بجمع المتغيرين ووضعهما في متغير ثالث اسمه

#include <iostream>

using namespace std;

int main(){

int x,y;

int sum;

cout << "frist Namber:";

cin >> x ;

cout << "second Namber:";

cin >> y ;

sum= x + y ;

return 0;

}

Output :

frist Namber:5

second Namber:10

sum is:15
